## 二、实现的功能

- 功能
    - [x] 有初始界面和设置界面
    - [x] 1v1地图
    - [x] 支持背景音效的滑动调节和打开关闭
    - [x] 支持多种类型的卡牌，包括战士，辅助，法师和射手
    - [x] 支持购买卡牌
    - [x] 支持卡池和刷新商店卡牌
    - [x] 支持卡牌升星功能
    - [x] 支持小小英雄的移动和血量展示
    - [x] 场上卡牌支持红蓝血条，辅助在蓝条满时可以释放技能
    - [x] 支持攻击动画特效
    - [x] 支持英雄自动寻路攻击，并有动画效果
    - [x] 支持练习模式（即单人模式），玩家可以和AI玩家对弈
    - [x] 支持联机模式，支持2个玩家联机对战
    - [x] 游戏结束可显示游戏胜利，失败
